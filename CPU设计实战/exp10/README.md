本实验新增了运算指令：
- 为新增指令添加译码逻辑
- 在ALU中新增了ANDN，ORN两种运算
- 新增乘法器、除法器和相关的控制信号`mul_div_op`，`mul_div_sign`。乘法和除法都在MEM级得到结果：乘法器在输入操作数的下一周期可以得到结果，故在EXE级无需阻塞；除法器在`complete`信号置起的下一拍得到结果，故在`complete`信号为高时可以`readygo`

注意事项：
- 模块级验证环境中复位信号为`resetn`，而CPU中为`reset`
- CPU中的除法器在第一拍置起`complete`，第二拍才会输出余数和商，目的是让MEM级收到运算结果，这点与模块级验证环境不同
- 乘法器时序较差，在50MHz下前递路径出现轻微的时序违约，但仍能通过上板测试
